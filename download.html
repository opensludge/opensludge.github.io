Other Distributors

Debian/Ubuntu

SLUDGE is available in Debian starting from Wheezy and in Ubuntu starting from 11.10 in the form of the packages sludge-engine, sludge-devkit and sludge-doc. If using older Debian/Ubuntu versions, you can manually download and install the packages (Debian, Ubuntu) after manually installing libalure1, version 1.1 (and, on Ubuntu 10.04 or older, libglee0d1).

Arch Linux

SLUDGE is available in the Arch User Repository.
